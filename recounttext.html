My Experience Creating a Website

Recently, I had an exciting experience creating my own website for a school project. At first, I felt confused because I did not understand how to design and code a website. However, I was very interested in learning new things, so I started studying HTML, CSS, and a little bit of JavaScript from the internet and YouTube tutorials.

The first step was planning the design of the website. I created a simple layout in Figma and chose colors, fonts, and images that matched the theme. After that, I started coding the website using HTML for the structure and CSS for the design. It was difficult at the beginning because I often made mistakes in the code, but I kept trying and fixing the errors.

One of the most enjoyable parts was making the website responsive so it could look good on both computers and mobile phones. I also added animations and interactive buttons to make the website more attractive. When the website was finally finished, I felt very proud and happy because I created it by myself.

From this experience, I learned that building a website needs creativity, patience, and problem-solving skills. It also improved my confidence and motivated me to learn more about web development in the future.

Every line of code I write today
is a step toward a better version of me tomorrow.
